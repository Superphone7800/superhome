Trivia:

  • The name and concept are based off of the 1979 glitch art piece of the same name, done by Jamie Faye Fenton, Raul Zaritsky, and Dick Ainsworth. The music a the end is sampled from this tape, which itself is from the Astrocade Demo cartridge. It's widely regarded as one of, if not, the first piece of glitch art.

 

  • This was first performed in Chicago in September 2025, which was actually the occasion for which I finished the lyrics. I realized months later but I realized that there was an alternate set of completed lyrics you can read here. I think the final ones are better but I can see what I was going for at least.
